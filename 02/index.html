Direct attention away from transaction statuses

Human attention span is shorter than the blockchain's update time

The user's attention is a precious resource. Embrace long confirmation times as inevitable, and de-emphasize transactions waiting to be cleared.

It's important to keep users informed about the system's status, but that doesn't mean transactions should steal the show. Never block the UI while waiting for the blockchain. Instead, consider these options:

Option A: status bar

Signing up for CryptoKitties is free and done off-chain. They ask the user's web3 provider to sign a message to prove it has the private key to his publicly-exposed Ethereum address.
